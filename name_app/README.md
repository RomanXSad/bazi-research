# Name App — Chinese Name Converter

Flutter app for converting Chinese names to pinyin, meanings, element analysis, and English name suggestions.

## Features

- Chinese → Pinyin conversion
- Character meaning lookup (388 characters)
- Element (五行) analysis with visual bars
- English name suggestions based on character meanings
- Auto-demo mode cycling through 10 example names

## Build

```bash
flutter pub get
flutter run          # iOS/Android/Web
flutter build apk    # Android APK
flutter build ios    # iOS IPA
```
